# My BS5839-1 Journey

This repository documents my hands-on learning and reflections while studying **BS5839-1 fire detection and alarm systems**

It captures my personal observations, technical insights, and practical experience gained from **training sessions and AutoCAD exercises**, where I practiced designing layouts and simulating device placements.

---

## What This Repo Contains

- **What Amazed Me (`Amazements.md`)**  
  Notes on aspects of BS5839-1 that fascinated me, including system design logic, protocols, and the precision behind device placement.

- **Misunderstood Parts (`Misconceptions.md`)**  
  Sections I initially found confusing and the reasoning behind them once I understood the theory and standards.

- **Field Perspective (`Implementation.md`)**  
  Insights from practical design exercises, highlighting challenges in adapting theoretical layouts to realistic constraints in AutoCAD.

- **Potential Tools (`Tools.md`)**  
  Suggestions for software tools that can assist in device layout, protocol validation, and fire simulation.

- **Commissioning (`Commissioning.md`)**  
  Observations on the commissioning process based on training exercises and how systems should be verified for correctness.

- **Checklist (`Checklist.md`)**  
  A structured, step-by-step checklist covering design, layout, and system verification based on BS5839-1.  
  - Includes a **“What I Gained”** section summarizing the learning and understanding obtained from following the checklist.

## Why I Created This Repo

This repository serves multiple purposes:  
- **Reflective learning:** Documenting my personal journey helps consolidate knowledge gained during training.  
- **Practical reference:** Provides a structured overview of BS5839-1 concepts and layout design practices.  
- **Portfolio showcase:** Demonstrates my technical understanding, practical skills in AutoCAD, and ability to critically reflect on industry standards.
